# Lunar Crew Builder

Статичный офлайн-сайт для Lunar: каталог карт, ростер-билдер и быстрый поиск по правилам.

## Запуск

Откройте `index.html` в браузере. Сборка и установка зависимостей не нужны.

Для локального сервера можно использовать:

```powershell
python -m http.server 8000
```

## Структура

- `index.html` - разметка SPA.
- `style.css` - весь интерфейс и адаптив.
- `data.js` - юниты, предметы, стартовый индекс правил.
- `script.js` - каталог, билдер, сохранение ростеров и поиск.
- `assets/cards/` - локальные PNG карт.
- `assets/rules/` - локальные PDF правил и сценариев.

## Что уже есть

- Каталог юнитов и предметов с фильтрами по типу, фракции, роли и поиску.
- Просмотр оригинальной PNG-карты в модальном окне.
- Билдер ростера NASA/USSR с предметами, подсчетом кредитов, массы и базовыми проверками.
- Сохранение ростеров в LocalStorage и экспорт TXT.
- Поиск по curated-индексу правил, способностям юнитов и текстам предметов.

PDF rulebook пока подключен как документ, но не проиндексирован OCR: в исходном файле нет надежного текстового слоя.
